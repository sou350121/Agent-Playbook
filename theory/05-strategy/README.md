# 05 — 戰略生存 (Strategy & Positioning)

> 技術能力是必要條件，但不是充分條件。
> 本模塊解決：在 AI 重塑一切的時代，工程師如何定位、如何變現、如何持續進化。

---

## 本模塊定位

這裡的「戰略」不是商業戰略課，而是**工程師的個人決策框架**：
- 哪些技能值得投資，哪些會被 Agent 替代？
- 如何把技術優勢轉化為可持續的收入？
- Agentic 時代的工程師角色如何演進？

## 核心觀點：Agentic 時代的工程師角色

```
被替代的：
  - 重複性代碼生成
  - 標準化文檔編寫
  - 已知問題的調試

不可替代的：
  - 系統設計與護欄設計
  - 需求轉化（模糊 → 可執行 Spec）
  - Agent 行為的評估與糾偏
  - 跨域知識的連接與判斷
```

工程師的核心競爭力不再是「寫代碼」，而是**設計讓 Agent 工作的系統**。
