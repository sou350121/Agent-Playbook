# AI 生态地图

> 快速定位 AI 工具生态，找到你需要的那个工具。

Pulsar 自动化系统每日扫描、分类、评级 AI 应用生态中的工具与趋势，结果汇聚在此目录。进来之前不需要知道自己要找什么——浏览索引，标签会引导你。

---

## 目录文件

| 文件 | 内容 | 更新方式 |
|------|------|----------|
| `app-index.md` | 125+ AI 工具注册表 | Pulsar 自动追加，禁止删改现有条目 |
| `technology-landscape.md` | Q1 2026 技术生态导览（5 层架构图） | 手动季度更新 |
| `influencers.mdx` | 中文 AI 圈关键人物目录 | 手动维护 |

### app-index.md

工具注册表，每条目包含：名称、类别、评级标签、一句话定位、链接。

**仅追加**：Pulsar pipeline 会在每日运行后自动写入新条目；人工编辑只能在文件末尾新增，不得修改或删除已有行。

### technology-landscape.md

从基础模型到最终用户应用的 5 层架构全景：

```
基础模型 → 推理/微调 → Agent 框架 → 应用层 → 交互界面
```

每层标注代表性工具与 2026 Q1 的移动方向。

### influencers.mdx

追踪对中文 AI 圈有实质影响力的人物：研究方向、核心观点、近期信号。
配合 Pulsar 社交情报模块使用效果最佳。
