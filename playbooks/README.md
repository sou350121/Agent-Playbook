# 执行手册

> 理解"怎么做"——从新手到生产级部署的完整路径。

本目录是 Agent-Playbook 的实战核心。每一篇都针对真实场景，每一个模板都经过验证。不需要从头读完——按你的角色和当前阶段选入口。

---

## 目录结构

### `onboarding/` — 入门学习路径

8 阶段结构化路径，从零开始到能独立部署：

```
心态建立 → 工具认知 → 第一个 Agent → 提示词调优
→ 多 Agent 协作 → 评估与迭代 → 生产部署 → 变现策略
```

附：**45 分钟快速通道**——时间紧迫时的最小可行学习路径。

### `tools/` — 工具集成指南

30+ 工具的具体接入方法，包括：

`Cursor` · `Claude` · `Docker` · `DashScope` · `GitHub Actions` · 及更多

每篇含：安装配置、核心用法、常见坑、与 Pulsar 系统的集成方式。

### `prompts/` — 提示词模板库

12 个精选模板，覆盖三类工作流：

| 类别 | 模板数 |
|------|--------|
| 设计（UI/系统架构） | 4 |
| 文案（技术写作/营销） | 4 |
| 开发（代码生成/审查） | 4 |

每个模板含变量说明和实际输出示例。

### `use-cases/` — 真实案例

已验证的落地案例，包括：

- AI 创业 180 天复盘
- 增长工程实战（从 0 到 PMF）
- Vibe Coding 生产环境部署全记录

### `security/` — 安全防护

重点：**IDE 自动执行风险**——当 Agent 拥有文件系统和终端权限时的攻击面分析与防护策略。

### `templates/` — 文档模板

开箱即用的文档脚手架：

- 代理章程（Agent Charter）
- 案例记录格式
- 失败日志模板（Failure Log）

---

## 按角色选入口

| 你是 | 推荐起点 |
|------|----------|
| 零经验，第一次接触 AI 工具 | `onboarding/` 第一阶段 |
| 开发者，想把 Agent 引入工作流 | `tools/` + `prompts/` |
| 创业者，评估 AI 落地可行性 | `use-cases/` + `onboarding/` 变现策略章节 |

---

## 通用路径

```
onboarding/（建立基础认知）
    ↓
tools/（按具体需求选读）
    ↓
use-cases/（找同类场景参考）
    ↓
templates/（启动自己的项目）
```
