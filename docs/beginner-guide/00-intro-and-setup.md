# 00 启程：认识 Agent 并装好你的“神笔”

在进入心法之前，我们先解决最基础的问题：**AI Agent 到底是什么？我该怎么把它装进电脑？**

## 1. 什么是 AI Agent (智能体)？

如果说普通的 AI（如网页版 ChatGPT）是一个**“博学的复读机”**，你问它答；
那么 AI Agent 就是一个**“有手有脚的数字员工”**。

```text
  [ 传统 AI ] : 只有大脑。你问它：“怎么写个网页？”，它给你一堆文字。
  [ AI Agent ] : 有大脑 + 有手脚。你告诉它：“帮我做个网页”，它会：
                1. 思考步骤 (计划)
                2. 新建文件夹 (动手)
                3. 写代码 (动手)
                4. 运行并检查错没错 (自我修正)
```

**一句话总结**：Agent 不只是会“说”，它更会“做”。

## 2. 第一步：装好 Cursor (目前最强的 AI 编辑器)

Cursor 是目前全世界最好用的 Coding Agent 载体，它是你的“底座”。

1.  **下载**：访问 [cursor.com](https://www.cursor.com/)，点击 **Download**。
2.  **安装**：像普通软件一样安装。如果你以前用过 VS Code，Cursor 会自动继承你所有的习惯。
3.  **登录与模型选择**：
    -   建议登录并开启 **Pro 试用**（通常有 2 周免费）。
    -   在设置中，确保模型选的是 **Claude 3.5 Sonnet** (目前最聪明的编程模型)。

## 3. 第二步：认识三个“必杀快捷键”

装好之后，你只需要记住这三个操作，你就已经赢了 80% 的小白：

| 快捷键 | 名称 | 干什么用 | 什么时候用 |
| :--- | :--- | :--- | :--- |
| **Ctrl + L** | **Chat (聊天)** | 问它问题，让它解释代码 | 当你想“问”的时候 |
| **Ctrl + K** | **Edit (行内编辑)** | 选一段话，直接改掉它 | 当你想“局部修改”的时候 |
| **Ctrl + I** | **Composer (全案编写)** | **最强功能**。让它写整个功能、建文件夹 | 当你想“指挥它干大活”的时候 |

## 4. 第三步：喂给它第一个指令

1.  在 Cursor 里按 **Ctrl + I**。
2.  在输入框里写：
    > “帮我写一个简单的网页，中间有一个红色的圆，点一下它会变大。”
3.  点击 **Submit**。
4.  看它怎么：**Plan (计划) -> Create (创建文件) -> Finish (完成)**。

## 5. 最小练习（3 分钟）
- 按照上面的步骤，让 Cursor 帮你生成那个“红色的圆”。
- 当它写完后，点击它给你的 **"Save All"**。

**当你看到文件夹里真的多出了文件，你已经跨过了 Agent 的门槛。**
