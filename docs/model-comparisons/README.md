## ⭐ 测评维度索引

- **逻辑深度**：能否理解复杂的业务因果链？
- **工程遵从度**：是否会私自修改你的 `STEERING.md` 或忽略 `AGENT.md`？
- **代码审美**：生成的代码是“能跑就行”还是具备工业级健壮性？
- **工具调用 (MCP)**：在复杂工具链下的响应成功率与容错性。
